About GNU

학생 성장과 지역혁신을 함께 만드는 경상국립대학교

경상국립대학교는 교육·연구·산학협력의 기반 위에서 창의적 인재 양성과 지역사회 발전을 목표로 하는 국립대학입니다. 본 페이지는 GitHub Pages 배포를 염두에 둔 정적 HTML 샘플입니다.
